# React Portfolio Template

A modern, responsive portfolio template designed for developers, designers, and creative professionals. Showcase your work with style and professionalism.

## Sections Included

- **Hero Section**: Personal introduction with call-to-action
- **About Me**: Detailed information about your background
- **Skills**: Technical skills with progress indicators
- **Projects**: Portfolio showcase with filtering and details
- **Experience**: Work history and achievements
- **Contact**: Contact form and social media links
- **Blog**: Optional blog section for articles

## Key Features

- **Responsive Design**: Looks great on all devices
- **Dark/Light Mode**: Toggle between themes
- **Smooth Animations**: Framer Motion for fluid interactions
- **SEO Optimized**: Built-in SEO features
- **Fast Loading**: Optimized for performance
- **Easy Customization**: Simple to modify and extend

## Technology Stack

- **React 18**: Latest React with concurrent features
- **TypeScript**: Type safety and better development experience
- **Tailwind CSS**: Utility-first styling framework
- **Framer Motion**: Animation library for smooth transitions
- **React Hook Form**: Form handling with validation
- **React Icons**: Comprehensive icon library

## Project Showcase

The portfolio includes a sophisticated project showcase with:

- **Grid Layout**: Responsive grid for project cards
- **Filter System**: Filter projects by technology or category
- **Modal Details**: Detailed project information in modals
- **Live Links**: Direct links to live projects and GitHub repos
- **Image Gallery**: High-quality project screenshots

## Getting Started

1. **Clone the Repository**:

```bash
git clone https://github.com/your-org/react-portfolio-template.git
cd react-portfolio-template
```

2. **Install Dependencies**:

```bash
npm install
```

3. **Customize Content**:

   - Update personal information in the data files
   - Add your projects to the projects array
   - Replace placeholder images with your own
   - Modify colors and styling as needed

4. **Run Development Server**:

```bash
npm run dev
```

## Customization Guide

### Personal Information

Update your personal details in the `src/data/personal.ts` file:

- Name and title
- Bio and description
- Contact information
- Social media links

### Projects

Add your projects in the `src/data/projects.ts` file:

- Project title and description
- Technologies used
- Live demo and GitHub links
- Project images

### Styling

Customize the appearance:

- Modify colors in `tailwind.config.js`
- Update fonts and typography
- Adjust spacing and layout
- Add custom animations
